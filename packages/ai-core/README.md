# @telli/ai-core

A package for API response standardization and utilities.

## Features

- Standardized API response types
- Response creation utilities
- Type guards for response validation
- Zod schema validation
- Takes care of Billing and Access control for API keys

## Installation

This package is part of the Telli monorepo and is not published separately.

## Scripts

- `pnpm test` - Run tests
- `pnpm test:watch` - Run tests in watch mode
- `pnpm test:coverage` - Run tests with coverage
- `pnpm lint` - Lint the code
- `pnpm types` - Check TypeScript types
